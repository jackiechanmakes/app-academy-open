Headings

Headings from 'h1' to 'h6' identify blocks and sections. Every page must have one and only one 'h1'. A website page will be penalized with lower search rankings if there isn't a 'h1' element.

Paragraphs

Paragraph of texts go in 'p' tags.

Lists

Unordered Lists

Unordered lists are marked with bullet points and go in 'ul' tags. Each list item goes in an 'li' tag.

Types of lists:

  • Unordered lists ('ul')
  • Ordered lists ('ol')

Ordered Lists

Ordered lists are numbered and go in 'ol' tags. Each list item goes in an 'li' tag.

Anchors (i.e. links)

Anchor tags create links to other pages. The URL goes in the 'href' attribute. Go to DuckDuckGo to search for more information.

Setting 'target="blank"' will open the link in a new window. See MDN for more details.

Anchor tags can also create links to a new location on the same page. Instead of putting a URL goes in the 'href' attribute, just use the 'id' of the element you want to navigate to. This allows you to Jump to the top of the page.

Text Emphasis

Italic

The 'em' element adds extra emphasis to the text within the opening and closing tags. This shows up as italic text in most browsers.

Bold

You can also use the 'strong' element to emphasize text. This shows up as bold text in most browsers.

Organizing content with 'div' and 'span'

'div'

The 'div' element is a block element, so each 'div' will show up on a new line. The 'div' element is the generic container for flow content. It has no effect on the content or layout until sytled using CSS.

'span'

The 'span' element is a generic inline element for phrasing content, so there will be no line break between each 'span'. It inherently does not represent anything. It can be used to group elements for styling purposes(using the class or id attributes), or because they share attributes, such as lang.

Breaking text using the 'br' tag

The 'br' element produces a line break in text. It is useful for writing a poem or an address, where the division of lines is significant.
